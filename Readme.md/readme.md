# Description

its is a simple project about a dino trying to catch some good fish inside the water and that will increase your score each time with 1 point , while eating the good fish he has many obstacles first there is a gun shoot coming from above he need to avoid if he got one shoot the lives decrease with one , he had 3 lives at the begenning of the game if he lose the 3 of them he will lose the game and appear game over , but don't worry you will have live chances appear on screen every 10 s but take care it will dissapeared after 4 second if you don't take it , if you take it it will add one life foe you , if you already have the 3 lives so it will increase your score with 5 points each time . enjoy the game for 60 second and then check your score and of course you can replay again the game ;)
